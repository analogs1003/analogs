# 単語でログを検索する

## 手法

sample030-000 形式（ログファイルそのまま）に対する検索手法

grep

ただし、単語完全一致検索なので（部分一致は含まないので）以下の 2 パターンの検索

- 単語の前後に半角スペースを入れて grep
- 行末は半角スペースを含んでいないので、単語の前にスペース、単語の後に行末で grep

## usage

```
$ ./sample040.sh word
```

## result

```
$ time ./sample040.sh QyRUZJtrm4 | wc -lc
```
